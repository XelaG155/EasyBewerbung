# Project Brief: EasyBewerbung

## Overview
EasyBewerbung is an intelligent job application assistant designed to help users optimize their job applications. The system analyzes job offers and adapts the user's CV, generates personalized motivational letters, and provides detailed company intelligence.

## Core Features
1.  **Job Offer Analysis**: Analyze job descriptions to understand requirements.
2.  **CV Adaptation**: Modify user's CV to match specific job offers based on facts from their uploaded documentation (CV, references, diplomas).
3.  **Match Scoring**: Provide a percentage match score between the user's profile and the job offer.
4.  **Document Generation**:
    *   Personalized Motivational Letter.
    *   Accompanying Letter/Email.
5.  **Company Intelligence**: Provide comprehensive data about the target company (activities, branches, earnings, etc.).
6.  **Multi-user System**: User accounts and management.
7.  **Credit System**: Pay-per-use model using credits.

## Phased Implementation

### Phase 1: Core System (MVP Start)
*   **Goal**: Store user documentation and capture job offers.
*   **Features**:
    *   User Document Storage (CV, References, Diplomas).
    *   Job Offer Link Input.
    *   Basic Text Extraction from documents.

### Phase 2: Analysis & Matching
*   **Goal**: Analyze content and provide feedback.
*   **Features**:
    *   AI Analysis of Job Offers.
    *   AI Analysis of User Documents.
    *   Match Percentage Calculation.

### Phase 3: Content Generation
*   **Goal**: Create tailored application materials.
*   **Features**:
    *   CV Adaptation (Fact-based rewriting).
    *   Motivational Letter Generation.
    *   Email Generation.

### Phase 4: Company Intelligence
*   **Goal**: Empower user with context.
*   **Features**:
    *   Company Data Aggregation.

### Phase 5: Platform Features
*   **Goal**: Commercialization and Scale.
*   **Features**:
    *   User Authentication.
    *   Credit/Payment System.
